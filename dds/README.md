# 打地鼠游戏 (Whack-a-Mole)

一个有趣的打地鼠小游戏，使用HTML、CSS和JavaScript制作。

## 游戏规则

- 点击"开始游戏"开始游戏
- 地鼠会在9个洞中随机出现
- 点击出现的地鼠可以获得10分
- 游戏时间为30秒
- 得分越高越好！

## 游戏特色

- 🎯 随机出现的地鼠
- ⏰ 30秒倒计时
- 🎨 精美的3D风格界面
- 📱 响应式设计，支持移动设备
- 🎉 智能评分系统

## 如何运行

### 方式一：使用启动脚本（推荐）

```bash
./start_game.sh
```

脚本会自动选择最佳的启动方式。

### 方式二：手动启动

1. **使用本地服务器（推荐）**：
   ```bash
   # Python 3
   python3 -m http.server 8000

   # 或者 Node.js (如果安装了)
   npx http-server

   # 然后在浏览器中访问 http://localhost:8000
   ```

2. **直接打开HTML文件**：
   - 双击 `index.html` 文件
   - 或者在浏览器中拖拽 `index.html` 文件

## 技术栈

- HTML5
- CSS3 (动画和响应式设计)
- JavaScript (ES6+ 类和异步编程)

## 文件结构

```
dds/
├── index.html    # 游戏主页面
├── style.css     # 游戏样式
├── script.js     # 游戏逻辑
└── README.md     # 说明文档
```

## 游戏控制

- **开始游戏**: 开始新游戏
- **重置游戏**: 停止当前游戏并重置所有状态
- **点击地鼠**: 点击出现的地鼠得分
 
## 使用自定义地鼠图片

如果你希望用自己的图片替换默认地鼠，请将两张图像放到 `dds` 目录下，并命名为：

- `mole1.png`
- `mole2.png`

脚本会在地鼠出现时随机选择其中一张显示。如果你把图片放在其他路径，请相应修改 `script.js` 中的图片路径。

> 另外脚本也会自动识别并使用 `head.png` 与 `seed.png`（若你已把这两张头像放在 `dds` 目录中），所以可以直接把你的头像重命名为 `head.png` / `seed.png` 放入目录。
