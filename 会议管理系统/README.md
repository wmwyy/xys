# 会议管理系统项目

## 📁 项目位置

所有会议管理系统相关文件已整理到 `会议管理系统/` 文件夹中。

### 文件夹内容：
- `meeting_signin_app.py` - 主应用程序
- `README.md` - 详细使用说明
- `start_app.sh` - 启动脚本
- `会 议 纪 要.docx` - 示例会议纪要文档

## 🚀 快速启动

进入会议管理系统文件夹：
```bash
cd "会议管理系统"
```

运行应用程序：
```bash
# 方法1：使用启动脚本
./start_app.sh

# 方法2：直接运行
python3 meeting_signin_app.py
```

## 📋 功能特色

- ✅ 会议信息录入和管理
- ✅ 参会人员签到跟踪
- ✅ 会议内容记录（主要内容、决定、任务计划）
- ✅ 实时预览会议纪要
- ✅ 导出双页Word文档（会议纪要+签到表）
- ✅ 数据保存和加载功能

*请在 `会议管理系统/` 文件夹中运行所有相关操作。*
