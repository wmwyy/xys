# 🎮 Streamlit游戏中心

这是一个基于Streamlit的游戏中心应用，包含贪吃蛇和打地鼠两个经典游戏，完全支持手机端触摸控制。

## 📱 支持的游戏

### 🐍 贪吃蛇游戏
- **电脑端控制**：使用方向键或WASD键控制蛇的移动，空格键暂停/继续
- **移动端控制**：点击屏幕相应区域控制方向，或使用界面上的方向按钮
- **特色功能**：分数统计、最高分记录、精美动画效果

### 🐭 打地鼠游戏
- **游戏规则**：30秒内点击出现的地鼠获得分数
- **控制方式**：鼠标点击或触摸击打地鼠
- **特色功能**：随机出现、分数统计、游戏结束评价

## 🚀 运行方法

### 快速启动（推荐）
```bash
./start_streamlit_app.sh
```

### 手动启动

#### 1. 安装依赖
```bash
python3 -m pip install -r requirements.txt
```

#### 2. 启动应用
```bash
python3 -m streamlit run streamlit_app.py
```

#### 3. 访问应用
打开浏览器访问显示的本地地址（如 http://localhost:8501）

## 📱 移动端使用

1. 在手机浏览器中访问应用地址
2. 选择想要玩的游戏
3. 享受触摸控制的游戏体验

## 🎯 技术特性

- **响应式设计**：完美适配桌面和移动设备
- **触摸友好**：专门为触摸设备优化的控制方式
- **离线可用**：游戏可在离线状态下运行
- **跨平台兼容**：支持所有现代浏览器

## 🔧 自定义

你可以轻松添加更多游戏到这个平台中，只需：

1. 在相应目录中创建游戏文件
2. 在`streamlit_app.py`中添加新的游戏页面函数
3. 在主页面中添加游戏选择按钮

## 📝 注意事项

- 确保所有游戏文件路径正确
- 图片资源需要放置在对应游戏目录中
- 移动端建议使用现代浏览器以获得最佳体验
