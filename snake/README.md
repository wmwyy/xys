# 贪吃蛇游戏

一个简单而有趣的贪吃蛇游戏，支持网页端和手机端。

## 功能特点

- 🎮 经典的贪吃蛇游戏玩法
- 📱 响应式设计，支持手机和平板
- ⌨️ 键盘控制（方向键或WASD）
- 👆 触摸控制（点击屏幕区域或虚拟按键）
- ⏸️ 游戏暂停/继续功能
- 🏆 分数系统和最高分记录
- 🎨 现代化的UI设计

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器运行（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000

   # 使用Node.js
   npx http-server

   # 然后在浏览器中访问 http://localhost:8000
   ```

## 游戏操作

### 键盘控制
- **方向键** 或 **WASD**: 控制蛇的移动方向
- **空格键**: 暂停/继续游戏

### 触摸控制
- **虚拟按键**: 在手机端显示虚拟方向按键
- **屏幕点击**: 点击蛇头周围的区域来控制方向

### 按钮控制
- **开始游戏**: 开始新游戏
- **暂停**: 暂停/继续游戏
- **重新开始**: 重置游戏状态

## 自定义图片（替换蛇头与食物）

如果你希望用自己的图片替换蛇头与食物，请将图片命名并放到 `snake/` 目录下：
- 蛇头：`head.png`
- 食物（种子）：`food.png`

图片会在游戏加载时自动读取；如果图片未就绪，游戏会使用默认的方块样式作为回退显示。

## 游戏规则

1. 使用方向控制键控制蛇的移动
2. 吃到食物（橙色方块）可以增长身体并得分
3. 避免撞到墙壁或自己的身体
4. 游戏结束后可以重新开始

## 技术实现

- **HTML5 Canvas**: 游戏渲染
- **CSS3**: 响应式布局和动画效果
- **Vanilla JavaScript**: 游戏逻辑
- **LocalStorage**: 最高分持久化存储
